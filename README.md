# Tea Cup Animation 🍵

A simple HTML/CSS animation of a tea cup with rising bubbles.

---

## 🚀 Features
- Smooth bubble animations using CSS keyframes
- Hover effect on the tea cup with scaling and shadow
- Gradient background for a cozy atmosphere

---

## 📂 Project Files
- `index.html` – main structure
- `style.css` – styling and animations
- `tea cup.png` – cup image asset
